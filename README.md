# Candle-Animation
This project is a creative candle animation built using HTML and CSS only. It visually represents two candles with animated facial expressions, flame effects, smoke, and background transitions. One candle appears to blow out the flame of the other candle, creating a playful and interactive animation effect.

The animation uses CSS keyframes, transforms, and transitions to simulate blinking eyes, moving flames, smoke effects, and dynamic body movements of the candles.

This project demonstrates how pure CSS animations can be used to create engaging visual effects without JavaScript.

# ⚙️ Technologies Used

• HTML5 – For structuring the candle elements.

• CSS3 – For styling and animations.

• CSS Keyframes – To create motion effects such as blinking eyes, fire movement, and smoke.

# ✨ Features

• Animated candle characters with eyes and facial expressions

• Blinking eyes animation

• Flame dancing animation

• Smoke effect when candle blows

• Background color animation

• Light wave glow effect

• Smooth CSS keyframe animations

• Fully built without JavaScript

# 🕯️ How the Animation Works

• The HTML file creates the structure of two candles, including eyes, mouth, flame, smoke, and sparkles.

• The CSS file controls:

  ° Candle body movement

  ° Blinking eye animation

  ° Flame dancing effect

  ° Smoke movement

  ° Background color transitions

  ° Light wave glow animation

Multiple CSS @keyframes animations are used to simulate realistic motion and interaction between the candles.

# 🚀 How to Run the Project

1. Download or clone the repository.

2. Open the project folder.

3. Run the index.html file in any web browser.

No additional setup or dependencies are required.
